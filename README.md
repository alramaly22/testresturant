# Marosh Bakery

Welcome to the Marosh Bakery frontend project! This repository contains the static website for Marosh Bakery, featuring our delicious menu of pizzas, manakish, appetizers, and more.

## Project Structure

This is a vanilla HTML, CSS, and JavaScript project. The file structure is straightforward:

- `index.html`: The main landing page.
- `menu.html`: The full menu overview.
- Category Pages: `pizza.html`, `manakish.html`, `appetizers.html`, `pidetr.html`, `kaak.html`, `breakfast.html`.
- `assets/`: Contains all static resources.
  - `css/`: Stylesheets including `styles.css`.
  - `js/`: Application logic `main.js` and `modal.js`.
  - `img/`: Images and icons used across the site.

## Setup and Usage

To view the project locally, you don't need any complex build tools.

1. Clone the repository.
2. Open the project folder in your code editor (e.g., VS Code).
3. Use an extension like **Live Server** to serve the `index.html` file, or simply drag and drop the `.html` files directly into your web browser.

## Technologies Used

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- [Remixicons](https://remixicon.com/) (For UI icons)
- [Swiper.js](https://swiperjs.com/) (For the popular items carousel)
- [ScrollReveal](https://scrollrevealjs.org/) (For on-scroll animations)
